# 定制主题

提示

votre-dieu 暂时只提供了一种定制主题的方式

  1. 克隆一份 votre-dieu 的源码
  2. 在根目录中 common 的文件夹中找到 const.scss 的文件
  3. 可以按照自己的业务，进行主题色的修改
  4. 下载 votre-dieu 所需的依赖
  5. 执行 yarn build 来进行打包
  6. 提交到公司的私有库或者 npm
